# 📁 Portfólio Front-end

Projetos de front-end desenvolvidos durante atuação em empresa de TI, utilizados em produção com clientes reais. Os dados foram anonimizados para fins de portfólio.

---

## 🗂️ Projetos

### 📊 Status Report — Sprint Tracker
Painel dark mode para acompanhamento de sprints e entregas de desenvolvimento.

**Funcionalidades:**
- Cards de resumo com contagem por status
- Tabela de atividades com filtro de busca em tempo real
- Pílulas de status com efeito neon animado (In Progress, Ready for Testing, Committed)

**Tecnologias:** `HTML` `CSS` `JavaScript`

---

### ✅ Checklist de Implantação de Portal
Checklist interativo para controle do processo de implantação de portais corporativos com clientes.

**Funcionalidades:**
- 8 fases de implantação com itens clicáveis
- Progresso salvo automaticamente via localStorage
- Sistema de modais com sub-itens detalhados por fase
- Campos de data integrados aos itens
- Barra de progresso geral

**Tecnologias:** `HTML` `CSS` `JavaScript`

---

### 📖 Guia de Implantação de Sistema
Documento interativo de onboarding para novos membros da equipe de implantação.

**Funcionalidades:**
- Visão geral em cards coloridos por etapa
- Tutorial passo a passo com "Como fazer" e "Pontos de atenção"
- Checklist interativo com progresso por seção
- Animações de scroll reveal
- Layout responsivo para mobile

**Tecnologias:** `HTML` `CSS` `JavaScript`

## 💡 Sobre

Esses projetos foram criados para uso interno e com clientes, com foco em **usabilidade**, **clareza visual** e **funcionalidade prática**. Nenhum framework foi utilizado — tudo feito com HTML, CSS e JavaScript puro.
